# price-paid-average

## setup

```sh
npm run start
```

## CRUD API

###CREATE
Creation of a company and it's stock records over 252 trading days.

```sh
/api/price
```

###READ
Get all companies and their stock records.
```sh
/api/price
```


###UPDATE
Update a company's information or stock record.

```sh
/api/price/:id
```

###DELETE
Delete a company and all of it's records.

```sh
/api/price/:id
```
